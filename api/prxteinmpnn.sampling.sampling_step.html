prxteinmpnn.sampling.sampling_step.sample_temperature_step(i, carry, decoder, neighbor_indices, mask, autoregressive_mask, model_parameters, hyperparameters=(1.0,))[source]#

Single autoregressive sampling step with temperature scaling.

Parameters:

  • i (int) – Current iteration.

  • carry (tuple[Union[Key[Array, ''], UInt32[Array, '2']], Float[Array, 'num_residues num_neighbors num_features'], Int[Array, 'num_atoms num_features'], Int[Array, 'num_residues'], Float[Array, 'num_residues num_classes'], Union[Array, ndarray, bool, number, Iterable[ArrayTree], Mapping[Any, ArrayTree], None]]) – Tuple containing current state (rng_key, edge_features, node_features, sequence, logits).

  • decoder (Callable[[tuple[Int[Array, 'num_atoms num_features'], Float[Array, 'num_atoms num_neighbors num_features'], Int[Array, 'num_atoms num_neighbors'], Int[Array, 'num_residues num_atoms'], Bool[Array, 'num_residues num_residues'], Int[Array, 'num_residues']]], Int[Array, 'num_atoms num_features']]) – Decoder function to update node features.

  • neighbor_indices (Int[Array, 'num_atoms num_neighbors']) – Indices of neighboring nodes.

  • mask (Int[Array, 'num_residues num_atoms']) – Atom mask for valid atoms.

  • autoregressive_mask (Bool[Array, 'num_residues num_residues']) – Mask for autoregressive decoding.

  • model_parameters (PyTree[str, 'P']) – Model parameters for the model.

  • hyperparameters (tuple[float | int | Array | GradientTransformation, ...]) – Hyperparameters for sampling. In this case, the temperature for scaling logits.

Return type:

tuple[Union[Key[Array, ''], UInt32[Array, '2']], Float[Array, 'num_residues num_neighbors num_features'], Int[Array, 'num_atoms num_features'], Int[Array, 'num_residues'], Float[Array, 'num_residues num_classes'], Union[Array, ndarray, bool, number, Iterable[ArrayTree], Mapping[Any, ArrayTree], None]]

Returns:

Updated carry state and None for scan output.

Example

sample_temperature_step, decoder=decoder, neighbor_indices=neighbor_indices, mask=mask, autoregressive_mask=autoregressive_mask, model_parameters=model_parameters, temperature=temperature,

) final_carry, _ = jax.lax.fori_loop(

0, iterations, sample_step, carry,
